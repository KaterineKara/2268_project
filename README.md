# 2268_project

В коде приложены тестирования методов для поиска мошеннических аномалий в финансовых данных.

Используемый датасет: IEEE-CIS Fraud Detection | Kaggle

Приведены:
- шаблон тестирования, содержащий полную предобработку датасета для работы, а также рекомендации для визуализации
- код для тестирования методов Логистическая регрессия, Байесовские сети, Random Forest, Isolation Forest, MLP, GNN, LSTM, GAN, DAMP.
